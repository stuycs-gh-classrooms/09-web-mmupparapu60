Koch Curve

Modifications:

  1. Every line that is made when depth is not one, color is randomized for that line.
  2. Made one extra right turn of 120 degrees at the end of the recursive code.

        def draw_koch(t, depth, length):
          t.pd()
          t.width(5)
          if depth == 1:
            t.fd(length)
            t.color("black")
          if depth > 1:
            t.pencolor((randint(0, 255), randint(0, 255), randint(0, 255)))
            draw_koch(t, depth - 1, length)
            t.lt(60)
            draw_koch(t, depth - 1, length)
            t.rt(120)
            draw_koch(t, depth - 1, length)
            t.lt(60)
            draw_koch(t, depth - 1, length)
            t.rt(120)

Sierpinski Triangle

Modifications:

  1. Length for all lines is length instead of length/2.
  2. The last angle is randomized to be either 60, 120, or 180.

        def draw_sierpinski (t, depth, length):
            t.pd()
            t.width(3)
            t.color("black")
            if depth == 1:
                t.fd(length)
                t.lt(120)
                t.fd(length)
                t.lt(120)
                t.fd(length)
                t.lt(120)
            else:
                draw_sierpinski(t, depth - 1, length)
                t.fd(length)
                draw_sierpinski(t, depth - 1, length)
                t.rt(180)
                t.fd(length)
                t.rt(120)
                t.fd(length)
                t.rt(60)
                draw_sierpinski(t, depth - 1, length)
                t.rt(120)
                t.fd(length)
                t.lt(random.randrange(1, 3) * 60)

Tree

Modifications:

  1. Deleted the last line of code that went backwards by length.
  2. Decreased the width of the branches as the tree forms.

        def tree (t, depth, length, angle, width):
            t.pd()
            t.width(width)
            if depth == 1:
                t.fd(length)
                t.bk(length)
            else:
                t.fd(length)
                t.rt(angle)
                tree(t, depth - 1, length, angle, width - 1)
                t.lt(angle * 2)
                tree(t, depth - 1, length, angle, width - 1)
                t.rt(angle)
